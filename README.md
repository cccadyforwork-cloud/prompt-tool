# 咖啡扇形滤纸提示词 MVP

这是一个本地静态 MVP，用来根据咖啡滤纸 SKU 和模板类型自动生成 Amazon 图组提示词。

## 使用方式

直接用浏览器打开 `index.html`。

功能：
- 选择 6 个 SKU 中任意一款。
- 选择参数规格、功能展示、综合卖点 3 个模板之一。
- 图片类型由模板决定，一次性输出完整图组提示词。
- 修改左侧已提取参数。
- 复制当前模板完整图组。

## 数据口径

- 采购单确认：V02 100/200、扇形02 100、扇形U02 200、扇形04 100/200。
- 1688 页面辅助：原木浆咖啡滤纸、V 形/扇形、盒装 SKU、可见单片规格、结构和适配信息。
- 竞品只用于参考图组逻辑和筛选卖点，不用于补写未确认参数。
- 单片尺寸优先；盒装 100/200 只作为包装数量，不作为产品尺寸。
- 资料没有明确参数的产品不借鉴、不参照，只写能提取到的规格。
- 场景展示模板后期补入。
