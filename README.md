# 通用产品提示词生成工具 MVP

这是一个本地静态 MVP，用来根据产品资料和模板类型自动生成 Amazon 图组提示词。当前内置咖啡滤纸作为演示数据，后续可扩展为通用产品资料提炼工具。

## 使用方式

直接用浏览器打开 `index.html`。

功能：
- 输入采购单 PDF、1688 HTML 文件、竞品链接 HTML 文件。
- 点击解析资料，从三个来源中提取产品名、SKU、包装数量、规格、材质、结构、适配和卖点线索。
- 选择解析出的当前产品 / SKU。
- 选择参数规格、功能展示、综合卖点 3 个模板之一。
- 图片类型由模板决定，一次性输出完整图组提示词。
- 修改左侧可替换参数，例如 Product Name、Material、Color、Structure、Selling Point。
- 占位符在页面中以蓝色显示，复制时保留纯文本提示词。
- 每张图可单独复制，也可复制当前产品完整图组。

## 数据口径

- 采购单用于确认产品款式、SKU、包装数量。
- 1688 HTML 用于辅助提取产品名、材质、规格、尺寸、结构和适配信息。
- 竞品 HTML 用于参考图组逻辑和筛选卖点，不用于补写未确认参数。
- 资料没有明确参数的产品不借鉴、不参照，只写能提取到的规格。
- 提示词中不展示冗长的 verified data 明细，只保留必要占位符。
- 场景展示模板后期补入。
