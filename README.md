# 咖啡扇形滤纸提示词 MVP

这是一个本地静态 MVP，用来根据 6 个咖啡滤纸 SKU 自动生成 Amazon 图片提示词。

## 使用方式

直接用浏览器打开 `index.html`。

功能：
- 选择 6 个 SKU 中任意一款。
- 选择 8 张图中的任意图片类型。
- 修改左侧占位参数。
- 复制当前提示词，或复制该 SKU 的 8 张图提示词。

## 数据口径

- 采购单确认：V02 100/200、扇形02 100、扇形U02 200、扇形04 100/200。
- 1688 页面确认：原木浆咖啡滤纸、V 形/扇形、亚马逊定制盒装 SKU。
- Amazon 参考页作图逻辑：白底主图、生活场景、尺寸箭头、适配场景、使用步骤、材质/过滤卖点。
- 单片尺寸优先；盒装 100/200 只作为包装数量，不作为产品尺寸。
- 未找到供应商明确单片尺寸的 SKU，使用相似品借鉴参数并保留占位符。
